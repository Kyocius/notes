## 卷积的性质

- 交换律
- 分配律：$f_1(t) \ast [f_2(t) + f_3(t)] = f_1(t) \ast f_2(t) + f_1(t) \ast f_3(t)$
- 结合律
- 时移性质
- 与冲激函数卷积：
  - $f(t) \ast \delta(t) = f(t)$
  - $f(t) \ast \delta'(t) = f'(t)$​
- 与阶跃函数卷积：
  - $f(t) \ast u(t) = \int_{-\infty}^t f(\lambda)d\lambda$
- 积分微分性质
